# DATA.md

## Назначение репозитория

Данный репозиторий демонстрирует реализацию учебной Retrieval-Augmented Generation (RAG) системы для поиска информации по каталогу фильмов и сериалов Netflix.

Проект предназначен для изучения полного цикла построения RAG-системы: от подготовки данных и построения индекса до получения ответа с указанием источников.

## Источник данных

Используется открытый датасет Netflix Titles.

Датасет содержит информацию о фильмах и сериалах, размещенных на платформе Netflix.

Размер корпуса:

* 8807 записей

## Используемые поля

Из исходного датасета используются следующие поля:

* title
* type
* release_year
* listed_in
* description

На основе этих полей формируется текст документа для последующей индексации.

## Что индексируется

Индексируются:

* название произведения;
* тип контента (Movie или TV Show);
* жанры;
* описание сюжета;
* год выпуска.

## Что не индексируется

Не индексируются:

* служебные идентификаторы;
* промежуточные файлы обработки;
* файлы индекса;
* исходный CSV-файл.

## Pipeline

```text
Netflix CSV
→ datasets.json
→ documents.jsonl
→ chunks.jsonl
→ TF-IDF Index
→ Retrieval
→ Answer Generation
→ Streamlit UI
```

## Метод поиска

Для поиска используется TF-IDF векторизация и косинусная близость между запросом пользователя и индексированными текстовыми чанками.

## Ограничения

Проект использует поиск по ключевым словам и не использует эмбеддинги или внешние LLM-модели.

Система работает только на локальном корпусе данных и не обращается к внешним API.
