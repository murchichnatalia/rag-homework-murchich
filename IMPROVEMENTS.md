# IMPROVEMENTS

## Реализованные улучшения

### 1. Отображение источников

Для каждого ответа пользователю отображаются документы-источники, их идентификаторы и значения similarity score.

### 2. Механизм отказа

Добавлен порог релевантности (threshold), который позволяет системе отказываться от ответа при отсутствии достаточного контекста.

### 3. Автоматическая подготовка данных

Реализован скрипт преобразования исходного CSV-файла Netflix в формат datasets.json.

## Возможные дальнейшие улучшения

### 1. Переход на Embeddings

Заменить TF-IDF на современные эмбеддинги (Sentence Transformers или OpenAI Embeddings).

### 2. Использование векторной базы данных

Добавить поддержку FAISS, ChromaDB или Qdrant.

### 3. Интеграция LLM

Подключить локальную или облачную языковую модель для генерации более естественных ответов.

### 4. Улучшение chunking

Использовать семантическое разбиение документов вместо фиксированного размера чанка.

### 5. Расширение корпуса данных

Добавить дополнительные источники информации о фильмах и сериалах, включая рейтинги, актерский состав и пользовательские отзывы.
