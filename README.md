# Netflix Titles RAG

Учебный Retrieval-Augmented Generation (RAG) проект, реализованный на основе каталога фильмов и сериалов Netflix.

Проект демонстрирует полный RAG pipeline:

данные → документы → чанки → TF-IDF индекс → поиск → генерация ответа → Streamlit UI.

## Используемые данные

В проекте используется открытый датасет **Netflix Movies and TV Shows**, опубликованный на платформе Kaggle.

Источник данных:

https://www.kaggle.com/datasets/shivamb/netflix-shows

Датасет содержит более 8 800 фильмов и сериалов Netflix с метаданными, включая название, тип контента, год выпуска, жанры, страну производства и текстовое описание.

Для построения RAG-системы использовались текстовые описания фильмов и сериалов. На их основе были сформированы документы, выполнено разбиение на чанки, построен TF-IDF индекс и реализован поиск релевантного контекста.

Размер корпуса после подготовки данных составляет 8 807 документов.

## Запуск проекта

Создание окружения и установка зависимостей:

```bash
uv sync
```

Построение индекса:

```bash
uv run python scripts/build_index.py
```

Запуск пользовательского интерфейса:

```bash
uv run streamlit run app/main.py
```
